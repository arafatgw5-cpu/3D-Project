# 🎴 3D Card Components Collection

A collection of beautiful, interactive **3D card components** with smooth animations and hover effects for React / Next.js applications.

## 🔗 Live & Repository

- 🌐 Live Site: https://3-d-project-six.vercel.app 
- 💻 GitHub Repo: https://github.com/arafatgw5-cpu/3D-Project  

---

## 📦 Components Included (4 Total)

### 1. 📸 Photo Card (`3d-card.jsx`)
Polaroid-style photo card with overlapping layout and animated grid background.

### 2. 🌀 3D Card Effect (`3d-card-effect.jsx`)
Interactive card with mouse-follow 3D transform and perspective animation.

### 3. ✨ Glass Card (3D)
Modern glassmorphism card with depth and hover lighting effect.

### 4. 🎯 Tilt Hover Card
Smooth tilt-based hover animation with shadow and scale effect.

---

## ⚙️ Tech Stack

- ⚛️ React / Next.js  
- 🎨 Tailwind CSS  
- 🧩 shadcn/ui (optional)  
- 🛠️ clsx + tailwind-merge

## 🚀 Installation

```bash
npx create-next-app@latest my-app
cd my-app

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

npx shadcn-ui@latest init
npm install clsx tailwind-merge
